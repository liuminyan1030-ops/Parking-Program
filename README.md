# Secure Parking Management System

## Project Overview
A Python-based utility for managing parking lot operations, focusing on capacity control and automated fee reporting.

## Key Features
* **Admin Authentication**: Secure login for managing system settings.
* **Automated Billing**: Calculates fees based on vehicle type and duration.
* **Persistent Records**: Daily charges and vehicle logs are saved to local text files.

## Tech Stack
* **Language**: Python
* **Concepts**: Conditional Logic, File Handling, Authentication
